# jsdoc README

快速为函数创建 JSDoc 风格的注释

## Features

### Generate JSDoc

选中一个函数的所有参数, 输入 `Generate JSDoc` 命令, 即可生成 JSDoc 风格的函数注释

## Requirements

无

## Extension Settings

此插件提供以下配置:

- `jsdoc.Author`: 生成 JSDoc 时将自动插入作者字段
- `jsdoc.EnabledDate`: 自动插入日期字段
